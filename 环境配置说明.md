# UAV Exif信息提取工具 - 环境配置与运行说明

## 项目简介

本项目是一个无人机（UAV）Exif信息提取工具，支持从照片和视频中提取GPS、姿态等数据，并生成多种格式的输出文件（Excel、CSV、XML、Shapefile）。

### 主要功能

- **功能1（photo）**：从JPEG照片中提取Exif信息（GPS坐标、姿态角度等）
- **功能2（video_snapshot）**：从视频中提取快照帧，并附加GPS信息
- **功能3（video_exif）**：从SRT文件中提取视频GPS轨迹数据

## 系统要求

- Python 3.9 或更高版本（推荐Python 3.10/3.11/3.12/3.13）

## 安装步骤

### 方法一：使用虚拟环境（推荐）

#### 1. 克隆或复制项目文件

将项目文件夹复制到本地，确保包含以下目录结构：
```
项目根目录/
├── src/
│   └── UAV_Exif_Tool_with_Spatial.py  # 主程序
├── 第一次无人机数据/    # 输入数据目录（可选）
├── 老师给的视频及信息/   # 视频数据目录（可选）
└── 输出结果/            # 输出目录（自动创建）
```

#### 2. 创建虚拟环境

打开命令行（Windows用户使用PowerShell或CMD），执行：

```bash
# 进入项目目录
cd 项目根目录

# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
```

#### 3. 安装依赖

在激活虚拟环境后，执行：

```bash
pip install -r requirements.txt
```

如果安装速度较慢，可以使用国内镜像源：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

#### 4. 运行程序

```bash
python src/UAV_Exif_Tool_with_Spatial.py
```

### 方法二：使用Anaconda

```bash
# 创建conda环境
conda create -n uav_exif python=3.10

# 激活环境
conda activate uav_exif

# 安装依赖
pip install -r requirements.txt

# 运行程序
python src/UAV_Exif_Tool_with_Spatial.py
```

## 输入数据准备

### 功能1（photo）输入
- 文件夹中包含`.jpg`格式的无人机照片
- 照片需包含GPS Exif信息

### 功能2（video_snapshot）输入
- 文件夹中同时包含`.MP4`视频和对应的`.SRT`文件
- SRT文件包含GPS轨迹数据

### 功能3（video_exif）输入
- 文件夹中包含`.SRT`文件
- SRT文件包含GPS轨迹数据

## 输出文件说明

程序运行后会生成以下文件（以"照片Exif信息"或"视频Exif信息"为前缀）：

| 文件格式 | 说明 |
|----------|------|
| .xlsx | Excel格式，包含所有数据列 |
| .csv | CSV格式，逗号分隔值 |
| .xml | XML格式，结构化数据 |
| .shp | Shapefile点要素，GIS数据 |
| .shx | Shapefile索引文件 |
| .dbf | Shapefile属性数据 |
| .prj | Shapefile坐标系定义（WGS84） |
| *_line.shp | 飞行轨迹线要素 |

## 界面说明

### 数据处理选项卡
- **选择功能**：photo / video_snapshot / video_exif
- **输入路径**：包含待处理文件的目录
- **输出路径**：结果文件的输出目录
- **快照帧数**：视频快照提取的帧数（30-50）
- **线程数**：并行处理的线程数

### 空间索引查询选项卡
- **K最近邻查询**：查找最近的K个点
- **范围查询**：查找指定半径范围内的点
- **查询参数**：经度、纬度、高度

## 常见问题

### 1. 安装依赖失败
- 确保Python版本符合要求（3.9+）
- 使用管理员权限运行命令行
- 更新pip：`python -m pip install --upgrade pip`

### 2. 程序无法启动
- 检查Python环境是否正确配置
- 确认已安装所有依赖：`pip list`
- 查看是否有错误信息提示

### 3. 无法生成Shapefile
- 检查输出目录是否有写入权限
- 确认输入数据包含有效的GPS坐标

### 4. 视频处理失败
- 确认OpenCV安装正确：`python -c "import cv2; print(cv2.__version__)"`
- 检查视频文件格式是否为MP4
